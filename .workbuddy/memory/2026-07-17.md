# 2026-07-17

## Komiku 漫画文件夹管理 Web 应用（最简版）
- 需求：基于 `du -sh --block-size=G /vol1/1000/Manga/*` 做文件夹大小可视化 + 删除 + 刷新。
- 实现：Python 标准库 http.server 单文件后端（零依赖）+ vanilla JS 前端。
  - `app.py`：GET /api/folders（du -sb 扫描，按字节精确，降序返回 JSON）、POST /api/delete（shutil.rmtree）、GET / 静态服务。
  - `static/index.html`：条形图按比例展示、二次确认 modal、刷新、错误提示、移动端适配。
- 改进点：原指令 --block-size=G 四舍五入到整 G 丢失分辨率，改用 `du -sb` 精确字节，前端换算 GB/MB/TB。
- 安全：删除接口 safe_target() 校验——禁 `/ \ ..`、realpath 越界检测，仅允许删 MANGA_DIR 直接子目录。已测 `../../etc`/`a/b`/`..` 均被拒。
- MOCK 模式（MOCK=1）：模拟数据，不执行 du、不删真实文件，用于本地预览。
- 配置：环境变量 MANGA_DIR/HOST/PORT/MOCK。
- 验证：Windows 本地 MOCK 模式端口 8090 跑通；语法检查 OK；中文删除（Python urllib 发 UTF-8）成功。
- 部署：上传 app.py + static/ 到 NAS，`python3 app.py`，访问 :8080。
- 注意：Git Bash 下 taskkill 的 `//F` 参数被 MSYS 路径转换破坏，杀进程用 PowerShell Stop-Process；curl 传中文 body 在 Windows Git Bash 会乱码，需用 Python urllib 或文件方式。
